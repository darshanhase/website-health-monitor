# Website Health Monitor (Beginner Flask Project)

A simple Flask app to check website health:
- Enter a URL
- Perform HTTP GET request
- Measure response latency
- Capture status code
- Store results in SQLite
- View current result and history

## Features
- Beginner-friendly code with comments
- Uses Flask, Requests, SQLite, HTML, Jinja templates
- No authentication, CSS frameworks, or JS frameworks

## Setup
```bash
pip install -r requirements.txt
python app.py
